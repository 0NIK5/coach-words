# CoachWords — Будущие функции

## Кнопка "Ещё раз" → улучшения

### 1. Отправка слова в конец очереди (Priority: Medium)
**Идея:** При нажатии "Ещё раз" слово перемещается в конец текущего списка слов на обучение, позволяя изучить другие слова перед повторением этого.

**Преимущества:**
- Естественный рабочий процесс обучения
- Интервал повторений остаётся без изменений
- Пользователь видит больше разнообразия слов в одной сессии

**Реализация:**
- Снять слово из текущей позиции
- Добавить в конец массива `localWords`
- Остаться на следующем слове (не увеличивая индекс)

**Статус:** Идея

---

### 2. Отметить статус "uncertain" (Priority: Low)
**Идея:** Создать промежуточный статус для слов, которые пользователь узнаёт нечасто. При нажатии "Ещё раз" слово отмечается как `uncertain` вместо полного сохранения.

**Преимущества:**
- Отслеживание сложных слов без полного их изучения
- Возможность показывать слова со статусом `uncertain` с более частыми интервалами

**Реализация:**
- Добавить тип статуса в `CardProgress`: `'uncertain'`
- При нажатии "Ещё раз" сохранить как `uncertain` с интервалом 1 день
- В квизе показывать эти слова наравне с `learning`

**Статус:** Идея

---

### 3. Переключатель режима "Спрятать перевод" (Priority: High)
**Идея:** При нажатии "Ещё раз" показать режим тестирования памяти — перевод и примеры скрыты, пользователь вспоминает значение без подсказок.

**Преимущества:**
- Активная практика отзыва перед запоминанием
- Повышает эффективность обучения
- Прямая обратная связь о том, помнит ли пользователь слово

**Реализация:**
- Добавить кнопку "🔄 Тест памяти" рядом с "Ещё раз"
- При нажатии показать режим "flip card": только слово видно
- Кнопка "Показать ответ" раскрывает перевод и примеры
- После раскрытия: кнопки "Запомнил" / "Знаю"

**Статус:** Идея

---

### 4. Счётчик повторений и пометка "сложное слово" (Priority: Medium)
**Идея:** Отслеживать, сколько раз пользователь нажимал "Ещё раз" на одном слове, и помечать часто повторяемые слова как "сложные".

**Преимущества:**
- Выявить проблемные слова
- Предложить альтернативное объяснение или дополнительные примеры для сложных слов
- Аналитика: какие слова сложнее всего запомнить

**Реализация:**
- Добавить поле `repeatCount` в `CardProgress`
- При нажатии "Ещё раз" → `repeatCount++`
- При `repeatCount >= 3` показать визуальный индикатор (🔴) — "сложное слово"
- Потом: предложить альтернативные примеры или мнемонику

**Статус:** Идея

---

### 5. Показать больше примеров (Priority: Low)
**Идея:** При нажатии "Ещё раз" открыть развёрнутую карточку с дополнительными примерами использования слова или синонимами.

**Преимущества:**
- Помогает увидеть слово в разных контекстах
- Облегчает запоминание через контекст

**Реализация:**
- Добавить кнопку "Показать ещё примеры" (зависит от наличия доп. данных)
- Добавить в `Word` поля: `example3`, `example3_ru`, `example4`, `example4_ru`
- Модальное окно с дополнительными контекстами

**Статус:** Идея

---

### 6. Система сложности (Difficulty Rating) (Priority: Medium)
**Идея:** Позволить пользователю указать, насколько сложно ему даётся слово (просто / нормально / сложно), что влияет на интервалы SM-2.

**Преимущества:**
- Персонализированные интервалы обучения
- Слова, которые даются легко, повторяются реже
- Сложные слова повторяются чаще

**Реализация:**
- При нажатии "Ещё раз" показать слайдер: 😊 Легко | 😐 Нормально | 😞 Сложно
- Сохранить `difficulty` в `CardProgress`
- Модифицировать SM-2: если `difficulty='hard'` → `easeFactor *= 0.8`, `interval *= 0.7`

**Статус:** Идея

---

### 7. Отправка в избранное / "Закрепить" (Priority: Low)
**Идея:** Добавить кнопку ⭐ для пометки слова как избранного или "закреплённого" для дополнительного внимания.

**Преимущества:**
- Быстрый доступ к важным словам в любой момент
- Фильтр в списке слов по избранному

**Реализация:**
- Добавить кнопку ⭐ на карточке слова
- Добавить поле `favorite: boolean` в `CardProgress` или отдельное хранилище
- На WordListScreen фильтр "Избранное"

**Статус:** Идея
